2D Materials for In-Memory & Neuromorphic Computing

As conventional computing approaches the limits of Moore’s Law, new paradigms such as in-memory and neuromorphic computing are gaining traction. Two-dimensional (2D) materials — atomically thin materials like graphene and transition metal dichalcogenides — are emerging as promising candidates for next-generation hardware.

Why 2D Materials?

Unique Physical Properties
2D materials offer several advantages for computing hardware:
  • Atomic-scale thickness → ultra-low power consumption
  • High carrier mobility → fast signal propagation
  • Tunable electronic properties → flexible device design
  • Compatibility with flexible and heterogeneous systems

In-Memory Computing

Breaking the Memory Bottleneck
Traditional architectures separate memory and computation, causing latency and energy inefficiency (von Neumann bottleneck).
  • 2D materials enable memristive devices
  • Support analog computation directly in memory
  • Reduce data movement and energy cost
These devices can perform matrix operations — critical for AI — directly where data is stored.

Neuromorphic Computing

Brain-Inspired Hardware
Neuromorphic systems aim to mimic biological neural networks:
  • Synaptic behavior using resistive switching devices
  • Plasticity mechanisms (learning in hardware)
  • Event-driven, energy-efficient computation
2D materials enable compact, scalable synaptic devices with tunable conductance states.

Key Device Types

Emerging Hardware Components
  • Memristors → Non-volatile memory + computation
  • Phase-change devices → Analog state transitions
  • Ferroelectric devices → Low-power switching
  • Transistor-based synapses → Multi-level learning behavior

Challenges

What Still Needs Work
  • Device variability and reliability
  • Scalability and fabrication challenges
  • Integration with CMOS technology
  • Standardized benchmarking methods

Summary

2D materials provide a promising pathway toward:
  • Energy-efficient in-memory computing
  • Hardware-based learning systems
  • Scalable neuromorphic architectures
They bridge the gap between materials science and next-generation AI hardware.
